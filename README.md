## 📝 Overview

This repository contains the frontend web development task submitted for the recruitment process of **Alcheringa**, the annual cultural festival of IIT Guwahati. The project is built from scratch using pure web technologies to demonstrate core competencies in structural design, responsive styling, and dynamic interactivity.

## 🚀 Technologies Used

* **HTML5**: For semantic and accessible page structure.
* **CSS3**: For custom styling, responsive layouts (Flexbox/Grid), and animations.
* **JavaScript (Vanilla)**: For DOM manipulation, event handling, and client-side logic.
